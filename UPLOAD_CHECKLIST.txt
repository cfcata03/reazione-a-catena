Before Starting:
  ☐ You have a GitHub account
  ☐ Git is installed (check: git --version)
  ☐ You know your GitHub username

Step 1: Configure Git (First Time Only)
  ☐ Set your name: git config --global user.name "Your Name"
  ☐ Set your email: git config --global user.email "your@email.com"

Step 2: Prepare Project
  ☐ Open Terminal
  ☐ cd /Users/alexcalin/Desktop/progetto
  ☐ Run: make clean
  ☐ Verify README looks good: open README.md

Step 3: Create GitHub Repository
  ☐ Go to: https://github.com/new
  ☐ Repository name: reazione-a-catena (or your choice)
  ☐ Description: Nuclear chain reaction simulator using multi-process IPC
  ☐ Choose Public or Private
  ☐ ⚠️ DON'T check "Initialize with README"
  ☐ Click "Create repository"
  ☐ Keep the page open!

Step 4: Get Personal Access Token
  ☐ Go to: https://github.com/settings/tokens
  ☐ Click "Generate new token (classic)"
  ☐ Name: "Mac Terminal Access"
  ☐ Check scope: "repo" (full control)
  ☐ Click "Generate token"
  ☐ COPY the token (save it somewhere safe!)

Step 5: Initialize Git
  ☐ git init
  ☐ git add .
  ☐ git commit -m "Initial commit: Chain reaction simulator"

Step 6: Push to GitHub
  ☐ Get commands from GitHub page (section: "push an existing repository")
  ☐ Replace YOUR_USERNAME in the command
  ☐ Run: git remote add origin https://github.com/YOUR_USERNAME/reazione-a-catena.git
  ☐ Run: git branch -M main
  ☐ Run: git push -u origin main
  ☐ Enter username when prompted
  ☐ Paste your token as password (NOT your GitHub password!)

Step 7: Verify Upload
  ☐ Go to: https://github.com/YOUR_USERNAME/reazione-a-catena
  ☐ You should see all files listed
  ☐ README should display below the files
  ☐ Badges should appear at the top

Step 8: Polish It (Optional but Recommended!)
  ☐ Click "⚙️" next to "About" on right side
  ☐ Add website: (leave blank or add link)
  ☐ Add topics: c, operating-systems, ipc, multi-processing, macos, fork
  ☐ Check "Include in home page"
  ☐ Save changes
  ☐ Star your own repository ⭐

Step 9: Share & Celebrate! 🎉
  ☐ Copy your repository URL
  ☐ Share with friends/classmates
  ☐ Add to your resume/CV
  ☐ Post on LinkedIn/Twitter

🆘 TROUBLESHOOTING:

Problem: "Permission denied"
  → Use Personal Access Token, not password

Problem: "Repository not found"
  → Check URL matches exactly
  → Make sure username is correct

Problem: "refusing to merge unrelated histories"
  → Run: git pull origin main --allow-unrelated-histories
  → Then: git push

Problem: Want to start over?
  → Run: rm -rf .git
  → Go back to Step 5
